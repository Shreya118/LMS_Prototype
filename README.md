# LMS Prototype

## Description

A simple prototype of a Learning Management System(LMS) where students can answer a
Pre-test, view their scores, then watch a video and answer a Post-test and can view its score

## Tech Stack

- React
- Vite
- CSS
- JSON (database)

## Commands to Run

npm install  
npm run dev
